## Sobre

O projeto é um exercício de JavaScript e CSS onde uma "magica" ocorre ao rolar a página. O efeito é bem divertido e foi muito interessante de ser feito. Position e a função scroll foram os assuntos mais presentes nesse exercício, muito bom.

O projeto foi Inspirado em um vídeo do canal Online Tutorials do Youtube.

## Tecnologias utilizadas

As seguintes e básicas tecnologias foram utilizadas no projeto:

- HTML e CSS
- JavaScript
- JQuery
